# Intelligent Business Analytics System

### Optimising Operations in a Computing Business Hub

Final year project — Department of Computer Science, Federal University of Technology Owerri.

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open **http://localhost:5000**

The database auto-seeds with 60 days of demo data on first run.

## Pages

| Page | URL | What It Does |
|------|-----|-------------|
| Dashboard | `/` | KPIs, revenue charts, hourly trends, recent insights |
| Analytics | `/analytics` | 60-day trends, service breakdown, customer metrics |
| Predictions | `/predictions` | 14-day demand & revenue forecasts |
| Insights | `/insights` | Automated decision-support recommendations |
| Data Entry | `/data-entry` | Record transactions, add customers |

## Tech Stack

Python • Flask • SQLite • scikit-learn • Chart.js • Bootstrap 5
